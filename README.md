# AxBoost v0.7.0

A non-root AxManager toolkit with reversible gaming/battery profiles, WebUI controls, device diagnostics, battery health, game management, and plugin conflict detection.

## New in v0.7.0

- Manual display controls: Auto/60/90/120 Hz peak request.
- Animation presets: off, 0.5x, and 1x.
- Window blur control.
- Android Battery Saver and Adaptive Battery Saver controls with state restoration when readable.
- Launch and force-stop configured games from WebUI.
- Combined diagnostic report (`axboost report`).
- Expanded WebUI Tools page.

AxBoost does not disable thermal protection, V-Sync, or write CPU/GPU governors.
